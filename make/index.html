writing it to a tag

  1. Install an NFC writer app — NFC Tools (iOS), NFC Tools (Android), or any equivalent.
  2. Copy the URL above.
  3. In the app, choose Write → URL/URI, paste, and tap your blank tag.

about the URL

The message lives in the URL fragment (after #). Browsers never send fragments to servers, so your message stays out of HTTP logs and analytics. Everything renders in the recipient's browser.
